# Protocols
- SSL/TLS libraries (OpenSSL) = Presentation Layer encryption (SSL/TLS display "HTTPS" in their URLs instead of "HTTP," indicating that communications are encrypted and secure.)
- Compression libraries (zlib, gzip) = Presentation Layer compression
- Serialization formats (JSON, Protocol Buffers) = Presentation Layer translation
- Character encoding (UTF-8) = Presentation Layer compatibility
