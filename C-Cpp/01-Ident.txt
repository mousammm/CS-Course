#1 Identifiers (int number = 10;)

- number is a identifer
- A-Z(upper/lower)
- 0-9
- ( _ )

- 1st letter cannot be number 

Variables:
    camelCase variable names (e.g., frequencyCount, personName).
    Constants UPPER_SNAKE_CASE (e.g., MAX_SIZE, PI).

Functions:
    camelCase function names (e.g., getName(), countFrequency()).
    Function names should generally be verbs or verb phrases that describe the action.

Structures:
    Use PascalCase for structure names (e.g., Car, Person).
    Structure names should be nouns or noun phrases.
