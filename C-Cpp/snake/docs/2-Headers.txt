Step3 : Main loop

int main() {
    Setup();  // Initialize everything
    while(!gameOver) {
      // The inside of this loop will be one "frame" of the game.
      Draw();
      Input();
      Logic();
   }
}

Every game loop needs to do three things:
    Draw the current state of the world.
    Get Input from the player.
    Update the game's Logic based on that input.
